<h3>This Python script provides a command-line interface TODO List functionality.</h3>

Supported commands:
<li>add "task"	        Add a new task</li>
<li>list	              List all tasks (show âœ… for done)</li>
<li>done <id>	          Mark task as completed</li>
<li>undone <id>	        Mark task as not completed</li>
<li>delete <id>	        Remove a task</li>
<li>clear	              Delete all tasks (ask for confirmation)</li>
<li>search "keyword"	  Show tasks containing keyword</li>
<li>stats	              Show number of done vs pending tasks</li>
<br/><br/>
Run with:

<code>python todo.py command [options]</code>
